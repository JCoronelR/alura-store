# alura-store
challenge de ML
### El propósito del análisis realizado.
El propósito del análisis realizado es ofrecer una base objetiva y fundamentada para tomar la mejor decisión sobre en qué tienda debe vender el Sr. Juan, considerando criterios clave del rendimiento comercial y la experiencia del cliente.​

### Objetivos del análisis
Comparar el desempeño de varias tiendas según métricas estandarizadas: ingresos totales, productos y categorías más vendidas, calificaciones promedio de clientes y costes de envío.​

Alinear la oferta del Sr. Juan con las preferencias de los consumidores y posicionarse estratégicamente frente a la competencia, maximizando el impacto y minimizando los riesgos.​

Este análisis permite tomar decisiones informadas y basadas en datos, replicando buenas prácticas, ajustando estrategias, y estableciendo un entorno propicio para el crecimiento en ventas y satisfacción del cliente final.​
